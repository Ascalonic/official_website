Barli

Barli is series of smartphone applications that focuses on changing the way we buy and sell vegetables for good. First app in the series is already complete and is about to start beta tests. The app targets Android 4.2+. This app instantly compares the prices, quality of vegetables, distance to shops around you wherever you are!

  • Client: You! :)
  • Date: June 2017
  • Technologies: LAMP stack, Android Front-End

Dogbuster for Appjar

How are we supposed to deal with the problem of Stray Dogs around us? Dogbuster does exactly that. This app lets to mark the location of stray dogs in the map, and anybody passing those location with this app running in background will get immediate alert! The project is complete and can be found at Google Play

  • Client: Appjar
  • Date: February 2017
  • Technologies: LAMP stack, Android Front-End

ZARS

ZARS stands for Zone Adaptive Response System. This project was done in the ARM Design Contest held by ACCS and TVS Motors in three stages - Bangalore (PESU), Chennai (IITM), Hosur (TVS). We won the third price at the national level. ZARS is an in-vehicle system that automatically recognise the traffic signs and displays before the driver in the console of the vehicle.

  • Date: September 2016
  • Technologies: Arduino (ARM IP), Android, IoT

FroSweet

FroSweet is an online photo styling web application which uses OpenCV for the image manipulation and uses concept of community computation for the processing task. This project later paved the way for Curev, an internal project to mine for customer reviews of a product from its photo. FroSweet is currently hosted Here. The project is currently inactive and is scheduled to be ported to a better form soon

  • Date: January 2017
  • Technologies: Image Processing, Web Development, .NET
